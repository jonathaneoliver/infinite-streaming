# Image Assets

This directory should contain the following image files for the Roku channel:

## Required Images

1. **logo_hd.png** (336x210 pixels)
   - Channel logo for HD menu focus

2. **logo_sd.png** (248x140 pixels)
   - Channel logo for SD menu focus

3. **splash_hd.png** (1920x1080 pixels)
   - Splash screen for HD resolution

4. **splash_sd.png** (720x480 pixels)
   - Splash screen for SD resolution

5. **channel_store_hd.png** (1280x720 pixels)
   - Channel store artwork (HD)

## Notes

For development and testing, you can create simple placeholder images with:
- Black background
- "InfiniteStream Player" text
- Appropriate dimensions for each image type

These images can be created using any image editing tool or generated programmatically.
