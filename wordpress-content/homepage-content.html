About Infinite Streaming

Infinite Streaming (from the InfiniteStream project) is a Docker-based HLS/DASH media server and AI no-code tool for testing video players under real-time, deterministic live-like conditions. It automatically generates low-latency HLS (LL-HLS) and DASH (LL-DASH) streams, including multiple segment variants, and features a powerful web dashboard for side-by-side comparisons, diagnostics, and monitoring.

Key Features

🤖 AI No-Code

Build and manage media streaming with minimal engineering effort. Automated stream generation and intelligent content management.

⚡ LL-HLS and LL-DASH

Real-time, low-latency stream generation for robust player testing. Support for both HLS and DASH protocols with sub-second latency.

🎬 Multi-variant Output

2s/6s segments, sliding window, and auto shutdown of idle sessions. Multiple quality levels and segment durations for comprehensive testing.

📊 Web Dashboard & Visualization

Mosaic views, playback comparisons, live offset tracking, and grid/multi-tile filtering. Real-time monitoring and diagnostics at your fingertips.

🔧 Diagnostics & Testing

Built-in failure injection (errors, timeouts, etc.), playback monitoring and diagnostics, session grouping, and traffic shaping for realistic test scenarios.

📁 Easy Content Management

Upload originals through the dashboard or copy directly to the host folder for on-the-fly streaming and testing. Simple drag-and-drop interface.

Use Cases

📹 Video Player Testing

Test video players under deterministic low-latency conditions with real-time feedback and metrics.

🔄 HLS/DASH Comparison

Compare HLS and DASH playback across different devices or player libraries side-by-side to understand performance differences.

🔍 Automated Diagnostics

Automated diagnostics, failure injection, and stress testing for stream resilience. Identify issues before they reach production.

🏭 QA and CI/CD Labs

QA and CI/CD lab environments for continuous integration of media streaming pipelines with automated test suites.

🌐 Network Throttling & ABR Testing

Grouped network throttling and patterned traffic shaping enables easy, side-by-side ABR (adaptive bitrate) comparison between different player builds, libraries, media segment durations, or encoding ladders. Benchmark impact on buffering, rebuffer events, and playback quality across multiple test conditions in parallel.

⚖️ ABR Algorithm Evaluation

Evaluate trade-offs in ABR algorithms by simulating real-world unstable networks with precise control over network conditions.

🎯 Repeatable, Deterministic Streaming

Traditional live streaming is not repeatable—every run is different due to network and source variability. Infinite Streaming solves this with deterministic, controlled content so you can re-run the same experiment for precise, apples-to-apples comparison (e.g., for ABR, CI/CD, and regression tests).

💾 Efficient Storage via Looping

Infinite Streaming lets you use short, looped segments as the basis of an "infinite" streaming session, minimizing storage requirements even for long-duration or many-parallel test scenarios.

🤖 Scriptable API

All network conditioning and fault injection actions (throttling, drops, errors, outages, custom network patterns) are fully scriptable and accessible via REST API—perfect for automated, repeatable QA or research workflows. Every test can be replayed with identical network and media dynamics.
